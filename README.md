## Instalacion del proyecto Laravel

1. hacer git clone con el link del proyecto
2. descargar las dependencias del proyecto con "composer i" & "npm i"
3. crear el archivo .env en el proyecto y copiar el contenido de ".env example"
4. modificar los datos de .env con los datos de la base de datos
5. ejecutar en la terminal php artisan migrate. | despues php artisan serve y npm run dev
